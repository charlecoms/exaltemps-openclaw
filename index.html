Session de TP : DevOps Agentique

OPENCLAW

Bienvenue dans l'ère de l'ingénierie autonome. Openclaw n'est pas un simple outil, c'est un agent capable d'orchestrer votre infrastructure via le langage naturel.

Retours d'expérience : quand l'IA s'emballe

Le "Flush" de Base de Données

Dans une Fintech, un agent mal configuré a interprété une commande de "nettoyage des logs" comme une instruction de purge complète d'un cluster RDS en production. Résultat : 4 heures d'interruption de service.

Le Déluge de Commits

En tentant de résoudre un conflit de fusion (merge), un agent a généré plus de 4 000 commits en 10 minutes, provoquant le blocage temporaire du compte GitHub de l'entreprise pour activité suspecte.
